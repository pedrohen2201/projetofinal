Sistema de Reserva de Salas de Aula

Descrição

Sistema textual (via terminal) que permite que professores façam
login/cadastro e reservem salas de aula, evitando conflitos de
horário. É possível cadastrar salas, fazer e cancelar reservas, e
consultar as próprias reservas.


Já existe um usuário padrão para testar:
- login: admin
- senha: admin123

Ou você pode se cadastrar como um novo professor na tela inicial.

Funcionalidade

Login/cadastro
- Cadastrar novo professor (login, senha e nome)
- Fazer login para acessar o sistema

Após o login
1. Listar salas cadastradas
2. Cadastrar nova sala
3. Reservar sala (verifica conflito de dia/horário; usa automaticamente o professor logado)
4. Listar todas as reservas
5. Ver minhas reservas
6. Cancelar uma das minhas reservas
